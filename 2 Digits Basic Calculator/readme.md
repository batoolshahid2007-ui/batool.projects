# Simple Python Calculator

A straightforward command-line calculator built with Python, designed to perform fundamental mathematical operations.

## Features
The program supports the following operations:
* **Addition (+)**
* **Subtraction (-)**
* **Multiplication (*)**
* **Division (/)**
* **Average (Mean of two numbers)**

## Getting Started
To run this program, ensure you have [Python](https://www.python.org/) installed on your machine.

### Instructions:
1. Clone this repository or download the `calculator.py` file.
2. Open your terminal or command prompt.
3. Navigate to the directory where the file is saved.
4. Execute the following command:
   ```bash
   python calculator.py
